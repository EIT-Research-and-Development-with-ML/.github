# .github
The Electrical Impedance Tomography (EIT) is a novel imaging technique that can show the Internal Conductivity Distribution (ICD) of a desired Region Of Interest (ROI) only with a current signal injection and boundary voltajes measurements.

This technique have several applications nowdays, mainly in the medical field. However, its real application for detecting diseases like breats cancer needs further research.

The repositories available here are aimed to improve the application of EIT for diseases detection a and even classification, using data processing and machine learning models.
